# Recipe-Grocery-AI-assistant

About The Project
-----------------------------

This is a multi-agent system where users can search for recipes and build out grocery lists for their meal plan.
There is a specialized agent that handles recipe discovery using natural language and specific criteria like protein and calories. There is also another specialized agent that handles the persistent grocery list that combines all your ingredient lists in one place.

Built With
-----------------------------
- Python
- LangChain
- LangGraph
- Streamlit
- Google Gemini (LLM)
